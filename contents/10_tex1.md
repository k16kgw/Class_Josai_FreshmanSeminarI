## TeXとは

TeXは，文書をきれいに組版するためのシステムである．
特に，数式を多く含む文書や，長いレポート，論文，書籍などの作成に用いられる．

Wordでは，画面上で文字や図を配置しながら文書を作る．
一方，TeXでは，文章と命令を書いたファイルを作成し，それを処理してPDFを作成する．

| ソフト | 特徴 |
| --- | --- |
| Word | 画面を見ながら文書を編集する |
| TeX | 命令を書いたファイルからPDFを作る |

TeXでは，見た目を直接動かすのではなく，文書の構造を命令として書く．

### TeXを使う場面

TeXは，次のような文書を作るときに便利である．

- 数式を多く含むレポート
- 卒業論文や研究論文
- 参考文献を含む文書
- 長い文書
- 同じ形式で何度も作成する文書

TeXは最初に覚えることが多いが，形式が整った文書を作りやすいという利点がある．

## 基本的なTeX文書

TeX文書は，次のような形で書く．

```latex
\documentclass[uplatex,dvipdfmx]{jsarticle}

\title{第11回 TeXによる文書作成1}
\author{学籍番号 氏名}
\date{\today}

\begin{document}

\maketitle

\section{はじめに}

本稿では，TeXによる文書作成の基本を学ぶ．

\section{本文}

TeXでは，文章と命令を組み合わせて文書を作成する．

\section{おわりに}

TeXを使うと，数式を含む文書を整った形式で作成できる．

\end{document}
```

### プリアンブル

`\begin{document}` より前の部分をプリアンブルという．
文書の種類，タイトル，著者名，日付などを設定する．

### 本文

`\begin{document}` から `\end{document}` までが本文である．
この範囲に，実際に出力したい文章を書く．

### 見出し

見出しは，次の命令で作成する．

```latex
\section{見出し}
\subsection{小見出し}
```

TeXでは，見出しの番号や余白を自動で整えてくれる．

## 箇条書き

TeXでは，箇条書きも命令で作成する．

### 記号付き箇条書き

```latex
\begin{itemize}
  \item Wordによる文書作成
  \item PowerPointによる発表資料作成
  \item TeXによる文書作成
\end{itemize}
```

### 番号付き箇条書き

```latex
\begin{enumerate}
  \item テーマを決める
  \item 構成を考える
  \item 本文を書く
\end{enumerate}
```

`\item` が箇条書きの1項目を表す．
`\begin{itemize}` と `\end{itemize}` の対応を間違えるとエラーになる．

---

## エラーへの対応

TeXでは，命令の書き間違いや記号の不足があるとPDFを作成できないことがある．

### よくあるエラー

- `\begin{document}` と `\end{document}` の対応がない
- `{` と `}` の数が合っていない
- 命令の先頭の `\` を忘れている
- ファイル名に空白や特殊文字が入っている
- 全角の記号を命令の一部に使っている

### 確認する順番

1. エラーが出た行の近くを確認する
2. `{` と `}` の対応を確認する
3. `\begin` と `\end` の対応を確認する
4. 直前に編集した箇所を確認する

エラーが出た場合は，慌てずに直前に変更した部分を見直す．
