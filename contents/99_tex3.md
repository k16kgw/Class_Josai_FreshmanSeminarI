まず，次のテンプレートをもとに文書を作成する．

```latex

\end{document}
```

## 数式を説明する

数式を含むレポートでは，式だけを並べてはいけない．
式の前後に，何を表す式なのかを文章で説明する．

### 例：2次関数

```latex
$a$，$b$，$c$ を実数とする．
2次関数は次の形で表される．
\[
f(x)=ax^2+bx+c
\]
ここで，$a \neq 0$ である．
```

### 例：和の公式

```latex
自然数 $n$ に対して，$1$ から $n$ までの和は次のように表される．
\[
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
\]
```

数式の前には「何について述べるか」を書き，数式の後には「式の意味」や「条件」を書く．

````{note} 演習2
次の2つの数式をTeXで入力し，それぞれの意味を本文で説明せよ．

```latex
\[
(x+y)^2=x^2+2xy+y^2
\]

\[
\int_0^1 x^2\,dx=\frac{1}{3}
\]
```
````

## 複数行の数式

長い計算は，複数行に分けると読みやすい．
`align` 環境を使うと，等号の位置をそろえられる．

```latex
\begin{align}
(x+1)^2
&= (x+1)(x+1) \\
&= x^2+2x+1
\end{align}
```

`&` は位置をそろえる場所を表す．
`\\` は改行を表す．

### 番号を付けない場合

数式番号が不要な場合は，`align*` を使う．

```latex
\begin{align*}
(x+1)^2
&= (x+1)(x+1) \\
&= x^2+2x+1
\end{align*}
```

## 表を含む文書

数学や情報のレポートでは，値や比較を表で示すことがある．

```latex
\begin{table}[h]
  \centering
  \caption{関数の値}
  \label{tab:function}
  \begin{tabular}{|c|c|}
    \hline
    $x$ & $f(x)=x^2$ \\
    \hline
    0 & 0 \\
    1 & 1 \\
    2 & 4 \\
    3 & 9 \\
    \hline
  \end{tabular}
\end{table}
```

本文中では，次のように説明する．

```latex
表\ref{tab:function} に，$f(x)=x^2$ の値を示す．
```

表を入れる場合は，表が何を示しているのか本文で説明する．

```{note} 演習3
次の値を表としてTeXで作成せよ．

| $x$ | $2x+1$ |
| --- | --- |
| 0 | 1 |
| 1 | 3 |
| 2 | 5 |
| 3 | 7 |

表にはキャプションとラベルを付け，本文中で参照すること．
```

## 図を含む文書

図を使う場合は，画像ファイルをTeXファイルと同じフォルダに置くと扱いやすい．
図にはキャプションとラベルを付ける．

```latex
\begin{figure}[h]
  \centering
  \includegraphics[width=0.6\linewidth]{sample.png}
  \caption{サンプル画像}
  \label{fig:sample}
\end{figure}
```

本文中では，次のように参照する．

```latex
図\ref{fig:sample} に，サンプル画像を示す．
```

図を入れる場合は，著作権や出典に注意する．
自分で作成した図であっても，何を示す図なのかを説明する．

## 参考文献と生成AIの利用記録

レポートを書くために参照した本やWebページがある場合は，参考文献を書く．

### 参考文献の例

```latex
\section*{参考文献}

\begin{itemize}
  \item 藤岡敦『手を動かしてまなぶ 微分積分』裳華房，2019年．
  \item 城西大学「図書館」https://libopac.josai.ac.jp/ （閲覧日：2026年7月13日）
\end{itemize}
```

### 生成AIの利用記録の例

```latex
\section*{生成AIの利用記録}

本レポートでは，生成AIを誤字脱字の確認に使用した．
AIの指摘を確認し，本文の誤字を修正した．
```

生成AIを使っていない場合も，次のように書く．

```latex
\section*{生成AIの利用記録}

本レポートでは生成AIを使用していない．
```
