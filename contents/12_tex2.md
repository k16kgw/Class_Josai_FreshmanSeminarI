## 数式を書く

TeXは，数式を含む文書を作成するのに適している．
数式には，文中に入れる数式と，独立した行に置く数式がある．

### 文中の数式

文中の数式は `$` と `$` で囲む．

```latex
関数 $f(x)=x^2$ を考える．
```

出力では，`$f(x)=x^2$` の部分が数式として表示される．

### 独立行の数式

重要な数式や長い数式は，独立した行に置く．

```latex
\[
f(x)=x^2+2x+1
\]
```

独立行の数式は，本文から分かれて表示されるため読みやすい．

### よく使う数式

| 入力 | 意味 |
| --- | --- |
| `x^2` | べき乗 |
| `a_i` | 添字 |
| `\frac{a}{b}` | 分数 |
| `\sqrt{x}` | 平方根 |
| `\sum_{i=1}^{n} i` | 総和 |
| `\int_0^1 f(x)\,dx` | 積分 |

## 数式を書くときの注意

数式だけを並べるのではなく，本文で記号の意味を説明する．

悪い例

```latex
\[
y=ax+b
\]
```

改善例

```latex
$a$ と $b$ を定数とする．このとき，直線の方程式は次のように表される．
\[
y=ax+b
\]
```

数式を使うときは，何を表す式なのかを本文で説明する．

## 表を作成する

TeXでは，`tabular` 環境を使って表を作成する．

```latex
\begin{tabular}{|c|c|}
\hline
項目 & 内容 \\
\hline
Word & 文書作成 \\
PowerPoint & 発表資料作成 \\
TeX & 数式を含む文書作成 \\
\hline
\end{tabular}
```

### 表の列指定

`{|c|c|}` の部分は，列の配置を指定している．

- `l`：左揃え
- `c`：中央揃え
- `r`：右揃え
- `|`：縦線

### 表の中の記号

- `&`：列の区切り
- `\\`：行の終わり
- `\hline`：横線

`&` や `\\` を忘れると，表が正しく作成できない．

## 図を挿入する

TeXで図を挿入するには，画像ファイルを用意し，`graphicx` パッケージを使う．

プリアンブルに次を追加する．

```latex
\usepackage[dvipdfmx]{graphicx}
```

本文では，次のように書く．

```latex
\begin{figure}[h]
  \centering
  \includegraphics[width=0.6\linewidth]{sample.png}
  \caption{サンプル画像}
  \label{fig:sample}
\end{figure}
```

画像ファイルは，TeXファイルと同じフォルダに置くと扱いやすい．

### 図の説明

`\caption{...}` で図の説明を付ける．
図を入れる場合は，本文中でもその図について説明する．

```latex
図\ref{fig:sample} に，サンプル画像を示す．
```

図だけを置いて説明しないと，読み手は何を見ればよいか分かりにくい．

## 相互参照

TeXでは，図や表にラベルを付け，本文中で参照できる．

### ラベル

図や表には，`\label{...}` でラベルを付ける．

```latex
\label{tab:software}
```

### 参照

本文中では，`\ref{...}` で番号を参照する．

```latex
表\ref{tab:software} に，各ソフトの用途を示す．
```

図や表を後から追加して番号が変わっても，TeXが自動で番号を調整する．
